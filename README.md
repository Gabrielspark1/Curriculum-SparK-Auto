# Currículum-SparK-Auto
Es una herramienta de automatización diseñada para crear currículums de alto impacto visual y profesional de forma 100% personalizada. A diferencia de las plantillas comunes, este sistema utiliza una interfaz en Kivy y un motor de generación en Python (FPDF2) para garantizar que cada CV sea único.
✨ Características Principales:
Escritura Auténtica: Incluye un sistema de "Bloqueo de Pegado" para fomentar que el usuario redacte su perfil de forma original y sincera.
Diseño SparK Signature: Marco exterior de identidad visual y firma discreta de diseño independiente.
Procesamiento en la Nube: Gracias a GitHub Actions, el proyecto convierte automáticamente tus datos en un PDF profesional listo para imprimir.
Modularidad: Separación inteligente de datos (Datos Personales, Formación, Aptitudes, Carta de Presentación) para una fácil edición.
🛠️ Tecnologías Usadas:
Python 3 (Motor principal)
Kivy (Interfaz de usuario móvil/escritorio)
FPDF2 (Generación de documentos PDF)
GitHub Actions (Automatización y despliegue)
