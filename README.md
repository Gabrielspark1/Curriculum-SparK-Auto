# Currículum-SparK-Auto

Es una herramienta de automatización diseñada para crear currículums de alto impacto visual...

## ✨ Características Principales:
* **Escritura Auténtica:** Incluye un sistema de "Bloqueo de Pegado"...
* **Diseño SparK Signature:** Marco exterior de identidad visual...
* **Procesamiento en la Nube:** Gracias a GitHub Actions...
* **Modularidad:** Separación inteligente de datos...

## 🛠️ Tecnologías Usadas:
* **Python 3** (Motor principal)
* **Kivy** (Interfaz de usuario móvil/escritorio)
* **FPDF2** (Generación de documentos PDF)
* **GitHub Actions** (Automatización y despliegue)
